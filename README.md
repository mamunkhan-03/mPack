A comprehensive learning project exploring modern Spring Boot technologies through multiple management modules (Employee, Student, Intern, etc.).
This is a hands-on learning project where I experiment with various Spring Boot technologies and best practices. Each module serves as a practical implementation to understand different concepts and patterns in enterprise backend development.

Currently Implemented:
Spring Boot 3.x ,
Spring Data JPA ,
Oracle Database ,
RESTful APIs ,
CORS Configuration

Learning & Exploring:
Redis (Caching) ,
Apache Kafka (Messaging) ,
Spring Security (JWT) ,
Docker ,
Swagger/OpenAPI
